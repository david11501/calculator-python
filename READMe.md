# Simple Python CLI Calculator

A lightweight, interactive command-line calculator written in Python that performs basic arithmetic operations on two numbers.

This project uses a dynamic function-mapping approach with a command loop to handle user commands at runtime.

## 🚀 Features

* **Basic Arithmetic:** Supports Addition, Subtraction, Multiplication, and Division.
* **Safety First:** Prevents crashes with input validation and zero-division handling.
* **Dynamic Updates (`change`):** Change the numbers at any point during runtime without restarting the program.
* **All-in-One (`all`):** View the results of all operations simultaneously.

---

## 🛠️ How to Run

1. Make sure you have **Python 3** installed on your system.
2. Clone this repository or download the script.
3. Open your terminal/command prompt, navigate to the folder, and run:

```bash
python 2_numbs_calc.py
```

(If you prefer the command `python calculator.py`, spune-mi și redenumez fișierul pentru tine.)

## Commands

- `multiply` — multiply the two numbers  
- `add` — add the two numbers  
- `difference` — subtract the second number from the first  
- `division` — divide the first number by the second  
- `all` — show results of all operations  
- `change` — change the two numbers  
- `help` — show this help  
- `quit` / `exit` — exit the program
